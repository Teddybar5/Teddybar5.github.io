GMTK 2023 Game Jam Submission: The Lonely Vampire

The theme for this game jam was "Roles Reverse". I decided to make a dungeon crawling game where the player is the monster at the end of the dungeon, but you're lonely so you want the adventurers to make it through your traps. Strategically activate traps to kill monsters and protect the adventurers from harm. The game loop consists of rining the "village bell" to summon a set number of adventurers and then, using the security cameras, activate traps in ways that protect the adventurers and kill the monsters attacking them. This project is unfinished but I am proud of the initial idea and demo project here.
